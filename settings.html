Scene Summariser Settings

General

Turns the extension on/off.
Planned hook to auto-run after sends (inactive for now).

Display Options

Template sent to the LLM. Variables: {{words}} (slider value), {{summary}} (current summary), {{last_messages}} (messages since last summary).
Older entries drop off when this limit is exceeded.

Prompt Injection

Writes the current summary into the extension prompt stack used at generation time.
Variables: {{summary}} (current summary), {{last_messages}} (messages since last summary), {{words}} (word limit).
Whether the injection participates in scanning (mirrors built-in memory behaviour).

Data Handling

When enabled, only messages after the last summary are sent to the main prompt (chat completion only).

Debug

Logs to console for troubleshooting.

Current Summary

Each click of Summarise Scene appends a new entry with an incremental Scene #.
